# 🚀 Tweetify

Fullstack Twitter clone with real-time features built using Node.js, React and React Native.

---

## 📌 Sobre o projeto

O **Tweetify** é uma aplicação fullstack inspirada no Twitter, desenvolvida durante a **Semana OmniStack 5 da Rocketseat**.

A aplicação permite que usuários publiquem tweets, visualizem uma timeline e interajam com posts em **tempo real**.

---

## 🧠 Funcionalidades

- ✅ Criação de tweets  
- ✅ Timeline em tempo real  
- ✅ Sistema de curtidas  
- ✅ Integração entre Web e Mobile  
- 🔄 Upload de imagens *(em desenvolvimento)*  
- 🔄 Autenticação com JWT *(em desenvolvimento)*  

---

## 🛠 Tecnologias utilizadas

### 🔹 Backend
- Node.js  
- Express  
- MongoDB  
- Mongoose  
- Socket.io  

### 🔹 Frontend (Web)
- React  
- Axios  

### 🔹 Mobile
- React Native

⚙️ Como rodar o projeto
🔹 Pré-requisitos
- Node.js instalado  
- MongoDB (local ou Atlas)  
- Yarn ou npm

---

🔹 Backend
```bash
-cd backend
-npm install
-npm run dev

🔹 Frontend
-cd frontend
-npm install
-npm start

🔹 Mobile
-cd mobile
-npm install
-npx react-native run-android

🔐 Variáveis de ambiente

Crie um arquivo .env na pasta backend:

MONGO_URL=sua_string_do_mongo
PORT=3000
